# Custom Apps Feature

The Custom Apps feature allows administrators to add external applications and tools to the EO-Toolkit catalog.

## Overview

Custom Apps enable:

- Integration of external tools
- Access to partner applications
- Extended functionality
- Resource sharing

## Adding Custom Apps

### Through Admin Panel

1. Log in as administrator
2. Navigate to Django Admin
3. Go to **Custom Apps**
4. Click **Add Custom App**
5. Fill in form:
   - **Name**: App name (unique)
   - **Description**: App description
   - **Link**: URL to application
   - **Thumbnail**: Preview image
   - **Keywords**: Comma-separated keywords
6. Click **Save**

### Required Fields

- **Name**: Unique identifier
- **Description**: What the app does
- **Link**: Valid URL
- **Thumbnail**: Image file
- **Keywords**: For search/filtering

## App Management

### Viewing Apps

- Browse in Custom Apps catalog
- Search by keywords
- Filter by keywords
- View app details

### Updating Apps

- Edit metadata in admin
- Update links
- Replace thumbnails
- Modify keywords

### Deleting Apps

- Delete from admin panel
- Thumbnail automatically removed
- App removed from catalog

## App Catalog

### Display

Apps appear in catalog with:

- Thumbnail image
- App name
- Description
- Keywords
- Direct link

### Access

Users can:

- Browse apps
- Search apps
- Filter by keywords
- Access external apps
- View app details

## Best Practices

1. **Descriptive Names**: Use clear, descriptive names
2. **Complete Descriptions**: Provide detailed descriptions
3. **Relevant Keywords**: Use appropriate keywords
4. **Quality Thumbnails**: Use clear preview images
5. **Valid Links**: Ensure links are working
6. **Regular Updates**: Keep information current

## Use Cases

- Partner tool integration
- External resource access
- Extended functionality
- Educational resources
- Complementary tools
