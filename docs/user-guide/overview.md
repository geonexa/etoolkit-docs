# User Guide Overview

Welcome to the EO-Toolkit User Guide! This comprehensive guide will help you make the most of all the features available in the platform.

## What is EO-Toolkit?

EO-Toolkit is a web-based GIS platform that combines Earth Observation data, AI-powered analytics, and interactive mapping tools. It's designed to support development projects by providing location-specific insights across various domains.

## Main Features

### Authentication & User Management
- User registration with email verification
- Secure login and password management
- User profile management

### Data Catalog
- Browse thematic Earth Observation datasets
- Access training materials and resources
- Explore case studies
- Discover custom applications
- View partner information

### Interactive Mapping
- Draw custom areas of interest
- Upload GeoJSON files
- Visualize multiple data layers
- Zoom, pan, and explore maps

### Analysis Tools
- Land Use Land Cover (LULC) analysis
- Precipitation analysis (CHIRPS)
- Actual Evapotranspiration (ETa)
- Climate change projections
- Water storage analysis (GRACE)

### Report Generation
- Generate comprehensive PDF reports
- Include multiple analyses
- Customize themes and project phases
- Download and share reports

### GeoServer Integration
- Upload spatial data layers
- Manage raster and vector data
- Style layers with SLD files
- Publish data for visualization

## Navigation Structure

The application is organized into several main sections:

```
Home

│   ├── Training Materials
│   ├── Case Studies
│   └── Custom Apps
├── Search/Recommendations
├── Area Management
├── Analysis Tools

```

## User Roles

### Regular User
- Access all public features
- Create and manage areas
- Run analyses
- Generate reports
- Browse data catalog

### Administrator
- All regular user features
- Access Django admin panel
- Manage GeoServer data uploads
- Configure system settings
- Manage users and permissions

## Best Practices

1. **Organize Your Areas**: Use descriptive names and organize by country
2. **Explore Before Analyzing**: Browse available data before running analyses
3. **Start Simple**: Begin with basic analyses before moving to complex ones
4. **Save Your Work**: Areas and analyses are saved automatically
5. **Check Data Availability**: Some analyses require specific data availability
6. **Review Reports**: Always review generated reports for accuracy
