    Data Catalog Guide

    The Data Catalog is your gateway to accessing Earth Observation datasets, training materials, case studies, and custom applications in EO-Toolkit.

    Overview

    The Data Catalog provides access to:

    • Thematic Data: Earth Observation datasets from Google Earth Engine and GeoServer
    • Training Materials: Educational resources, tutorials, and guides
    • Case Studies: Real-world application examples
    • Custom Apps: External applications and tools
    • Partners: Information about data providers and partners

    Accessing the Catalog

    1. Log in to your account
    2. Click Data Catalog in the navigation menu
    3. Select the catalog type you want to browse:
    4. Thematic Data
    5. Training Materials
    6. Case Studies
    7. Custom Apps

    Browsing Data

    Thematic Data Catalog

    The Thematic Data catalog includes:

    • GEE Datasets: Google Earth Engine datasets
    • GeoServer Layers: Uploaded spatial data layers
    • Metadata: Descriptions, providers, and keywords
    • Thumbnails: Visual previews of datasets

    Viewing Dataset Details

    1. Click on any dataset card
    2. View detailed information:
    3. Dataset name and description
    4. Data provider/partner
    5. Keywords and categories
    6. Spatial extent
    7. Temporal coverage
    8. Usage instructions

    Viewing Data on Map

    1. From the dataset details page
    2. Click View on Map
    3. The dataset will be loaded in the map viewer
    4. Use map controls to explore the data

    Training Materials

    Browse educational resources:

    • Categories: Filter by topic
    • Sources: Filter by provider
    • Languages: Filter by language
    • Search: Find specific materials

    Case Studies

    Explore real-world applications:

    • Publishers: Filter by organization
    • Types: Filter by study type
    • Year: Filter by publication year
    • Search: Find relevant case studies

    Custom Apps

    Access external applications:

    • Keywords: Filter by topic
    • Search: Find specific apps
    • Links: Direct access to applications

    Search and Filters

    Using Search

    1. Enter keywords in the search box
    2. Search matches:
    3. Dataset titles
    4. Descriptions
    5. Keywords
    6. Results update automatically

    Applying Filters

    Filters vary by catalog type:

    Thematic Data: - GEE Categories - Theme - Project Phase - Partner/Provider

    Training Materials: - Category - Source - Language

    Case Studies: - Publisher - Type - Issued Year

    Custom Apps: - Keywords

    Using Filters

    1. Expand filter sections
    2. Select one or more filter options
    3. Results update automatically
    4. Clear filters to reset

    Pagination

    • Results are paginated (30 items per page)
    • Use navigation arrows to browse pages
    • Page numbers show current position

    Dataset Information

    Dataset Cards Display

    Each dataset card shows:

    • Thumbnail: Visual preview
    • Title: Dataset name
    • Description: Brief overview
    • Provider: Data source
    • Keywords: Relevant tags
    • Upload Date: When added to catalog

    Detailed View

    Clicking a dataset opens detailed information:

    • Full description
    • Technical specifications
    • Usage guidelines
    • Spatial and temporal coverage
    • Related resources

    GeoServer Datasets

    GeoServer datasets are uploaded by administrators and include:

    • Raster Data: GeoTIFF files
    • Vector Data: Shapefiles
    • Styles: SLD styling files
    • Thumbnails: Preview images

    Viewing GeoServer Layers

    1. Select a GeoServer dataset
    2. Click View on Map
    3. Layer loads with:
    4. WMS tile URL
    5. Spatial extent
    6. Legend information
    7. Style information

    Recommendations

    The Search/Recommendations feature provides:

    • Theme-based: Filter by development theme
    • Phase-based: Filter by project phase
    • Country-specific: Recommendations for selected countries
    • Contextual: Relevant data, training, and case studies

    Using Recommendations

    1. Navigate to Search page
    2. Select:
    3. Theme (e.g., Agriculture, Water, Climate)
    4. Project Phase
    5. Country
    6. View recommended:
    7. Datasets
    8. Training materials
    9. Case studies
    10. Standard Operating Procedures (SOPs)

    Best Practices

    1. Use Filters: Narrow down results efficiently
    2. Read Descriptions: Understand data before using
    3. Check Coverage: Verify spatial and temporal coverage
    4. Review Metadata: Check data quality and sources
    5. Save Favorites: Bookmark useful datasets
    6. Explore Related: Check related resources

    Tips

    • Keywords: Use specific keywords for better results
    • Multiple Filters: Combine filters for precise results
    • Clear Filters: Reset to see all available data
    • Export Lists: Note down useful dataset IDs
    • Check Updates: Catalog is regularly updated

    Troubleshooting

    No Results Found

    Solutions: - Clear filters and search terms - Check spelling - Try broader search terms - Verify catalog type selection

    Dataset Not Loading

    Solutions: - Check internet connection - Verify dataset availability - Try refreshing the page - Contact support if issue persists

    Map Not Displaying

    Solutions: - Check browser console for errors - Verify GeoServer connection - Ensure area of interest is selected - Try a different dataset
